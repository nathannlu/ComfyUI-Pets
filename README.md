## ComfyUI Pets Installation

1. `cd custom_nodes`
2. `git clone https://github.com/nathannlu/comfyui-pets.git`
3. In your ComfyUI, you can 'right click' and select 'Add Node'. Find 'Pets' > 'Comfy Pet'.
4. Enjoy your new buddy
